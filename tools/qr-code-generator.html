About QR Code Generator

Create QR codes instantly in your browser. Supports URLs, plain text, email addresses, WiFi credentials, phone numbers, and SMS. All processing happens client-side — no data is sent to any server.

Supported QR Code Types

  • Text/URL: Any text or web address — scanning opens the URL in a browser
  • Email: Pre-filled email with recipient, subject, and body
  • WiFi: Instantly connect to a WiFi network by scanning (WPA/WPA2/WEP)
  • Phone: One-tap dialing from any phone
  • SMS: Pre-filled SMS messages

Error Correction Levels

  • Low (7%): Smallest QR code, minimal damage tolerance
  • Medium (15%): Good balance of size and reliability (default)
  • Quartile (25%): Higher reliability, slightly larger code
  • High (30%): Maximum reliability — code works even if 30% is damaged

FAQ

What size should I use?

300×300 works for most uses (screens, documents). Use 400+ for print materials. The QR code is vector-based internally, so larger sizes stay crisp.

Can I customize colors?

Yes! Use the foreground and background color pickers. Ensure high contrast for reliable scanning — dark foreground on light background works best.

Do QR codes expire?

No. QR codes generated here are static — they encode the data directly. They will work forever as long as the encoded URL or resource is still available.
