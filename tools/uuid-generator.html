About UUIDs

A UUID (Universally Unique Identifier) is a 128-bit number used to uniquely identify resources. UUID v4 uses random numbers, making collisions astronomically unlikely.

UUID Format

Standard format: xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx where x is a random hex digit and y is one of 8, 9, a, or b.

Common Use Cases

  • Database primary keys
  • API request/response identifiers
  • Session tokens
  • File naming to avoid collisions
  • Distributed system identifiers
