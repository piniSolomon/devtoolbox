About Favicon Generator

A favicon (favorites icon) is the small icon displayed in browser tabs, bookmarks, and search results. Modern websites need multiple sizes for different devices and platforms.

Required Favicon Sizes

  • 16×16: Classic favicon.ico size (browser tabs, bookmarks)
  • 32×32: Modern browser tabs and taskbar
  • 180×180: Apple Touch Icon (iOS home screen)
  • 192×192: Android Chrome (web app manifest)
  • 512×512: PWA splash screens and high-DPI displays

FAQ

What format should my favicon be?

Use PNG for modern browsers (best quality, transparency support). ICO is only needed for very old IE support. SVG favicons are supported in modern browsers and scale perfectly.

Where do I put the favicon files?

Place them in your site's root directory and add the link tags to your HTML head. The classic favicon.ico should be in the root; other sizes can be anywhere referenced by the HTML tags.
